# Retail Inventory Analysis

SQL + Excel analysis of retail pharmacy inventory and sales data to uncover operational insights, inventory inefficiencies, and profit opportunities.

## Project Overview

This project demonstrates how SQL and Excel can be used to analyze retail inventory data and uncover business insights. The analysis focuses on identifying sales trends, inventory performance, and opportunities to improve operational efficiency.

## Business Questions

- Which products generate the highest revenue?
- Which products have the highest profit margins?
- Which items are overstocked or understocked?
- Which categories experience the highest demand?
- What inventory trends could impact profitability?

## Analyst's Thought Process

This project approaches inventory analysis from a business operations perspective rather 
than simply reporting numbers. The objective is to identify patterns, investigate the 
factors driving inventory performance, and translate data into practical recommendations 
that support operational efficiency and profitability.
